MonsterDun

A 2D top-down shooter game developed with Unity, where players fight through waves of enemies, collect energy resources, and defeat a powerful boss to complete the mission.

Features
Player movement and combat system
Shooting, ammo, and reload mechanics
Multiple enemy types with unique behaviors
Boss battle with special attack patterns
Enemy spawning and progression system
Health, damage, and collectible systems
Game state management (Pause, Win/Lose, Restart)
Object Pooling system for bullet optimization
Gameplay UI (HP Bar, Ammo Counter, Energy Tracker)
Technologies
Unity
C#
Object-Oriented Programming (OOP)
Object Pooling
Unity UI
Audio System
Git
Platforms
Windows
Project Structure
Player System: Movement, combat, health management.
Enemy System: Reusable enemy architecture using inheritance and polymorphism.
Boss System: Multi-skill boss encounter triggered by player progression.
UI System: Gameplay HUD and game state screens.
Optimization: Bullet Object Pooling to reduce runtime instantiation.
Learning Outcomes
Applied OOP principles (Inheritance, Encapsulation, Polymorphism) in game development.
Built modular gameplay systems that are easy to extend and maintain.
Improved performance using Object Pooling techniques.
Developed a complete gameplay loop from combat to victory conditions.
